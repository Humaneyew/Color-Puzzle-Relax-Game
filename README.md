Color Puzzle: Relax Game — це мінімалістична гра-релакс, у якій гравець відновлює гармонію кольорів, впорядковуючи відтінки у правильному порядку.
Проста механіка, плавна анімація та спокійна атмосфера роблять цю гру ідеальною для коротких сесій відпочинку.

## Технологічний стек

Проєкт зібрано на Flutter 3 та орієнтований на Material 3. Для керування станом використовується пакет `provider`, а для звукового супроводу — `audioplayers`. Аналіз коду виконується завдяки `flutter_lints`, модульні тести запускаються стандартною командою `flutter test`.

## Структура каталогу `lib/src`

- `components/` — реюзабельні віджети інтерфейсу та тема Material 3 (`app_theme.dart`).
- `screens/` — екрани застосунку.
- `logic/` — контролери і сесійна логіка гри (`game_board.dart`, `game_session.dart`).
- `logic/services/` — обгортки над платформеними сервісами (звук, реклама).
- `data/models/` — типи даних домену (`level.dart`, `tile.dart`, `game_result.dart`).

### Основні моделі

- `GradientPuzzleLevel` — опис рівня, включно з палітрою, розміром сітки та фіксованими плитками.
- `GradientTile` — модель окремої плитки з кольором і правильним індексом.
- `GameResult` — агрегує статистику проходження рівня (ходи, підказки, тривалість).
- `GameSession` — відповідає за прогрес гравця, життя, підказки та історію результатів.
